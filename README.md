# air-quality-index-dashboard
# 🌍 Air Quality Index (AQI) Dashboard

## 📌 Overview
This project presents an interactive Air Quality Index (AQI) Dashboard built using Power BI to analyze air pollution across different regions in India. It provides insights into AQI trends, pollutant distribution, and city-level air quality conditions.

---

## 🎯 Objectives
- Monitor air quality across states and cities
- Identify pollution trends over time
- Analyze dominant pollutants affecting AQI
- Support data-driven environmental decisions

## 📊 Key Insights
- PM10 and PM2.5 are the most dominant pollutants
- Majority AQI values fall under "Poor" and "Moderate" categories
- Certain cities consistently show higher AQI levels
- Seasonal fluctuations significantly impact AQI trends

---

## 🛠️ Tools & Technologies
- Power BI – Data visualization and dashboard creation  
- Excel / CSV – Data storage and preprocessing  
- Python (optional) – Data cleaning and transformation  

---

## 📂 Dataset
The dataset includes:
- AQI values
- Pollutant levels (PM2.5, PM10, NO2, SO2, CO, O3)
- Date and time
- Location (state and city)

Data Source: CPCB / Open datasets (Kaggle)

---

## ⚙️ ETL Process
1. Extract – Collected AQI data from dataset/API  
2. Transform – Cleaned missing values, handled inconsistencies  
3. Load – Imported processed data into Power BI  

---

## 🚀 How to Use
1. Open the `.pbix` file in Power BI  
2. Refresh dataset if needed  
3. Use filters (state, city, pollutant)  
4. Explore trends and insights
